# LONLY HARNESS Documentation Webpage

This directory contains a standalone, production-grade documentation webpage for the [LONLY HARNESS](https://github.com/poppp334/Lonly_HARNESS) project — an autonomous AI-powered penetration testing agent for Kali Linux.

## Contents

| File | Description |
|------|-------------|
| `index.html` | Full documentation webpage (single-file, embedded CSS/JS) |
| `assets/logo.svg` | LONLY vector logo with animated shield design |

## Sections

1. Hero / Header — Animated typing effect, CTA buttons, SVG logo
2. Legal Disclaimer — Prominent warning box
3. Overview — ReAct loop explanation + architecture diagram
4. Tech Stack — 8-layer technology table
5. Key Features — 9 glass-morphism feature cards
6. Tool Arsenal — 24 tools across 5 categories in interactive tables
7. Performance Comparison — Animated progress bars, comparison table, bar chart, methodology box
8. Installation & Quick Start — Copyable terminal blocks
9. System Requirements — System + Kali tools lists
10. Footer — GitHub link, disclaimer reminder

## Usage

Open `index.html` in any modern browser. No build step or server required — fully self-contained.

## Design

- Dark theme with cybersecurity red accents (#ff3333, #cc0000)
- Matrix green (#00ff41) for code/terminal elements
- Glass-morphism cards, gradient borders, parallax background
- JetBrains Mono for code, Inter for body text
- Fully responsive (mobile, tablet, desktop)
