Overview

What is LONLY HARNESS?

An interactive, autonomous penetration testing AI agent that runs entirely on your local machine.

LONLY (Logically Optimized Network Logistics & Intelligence) uses a ReAct (Reasoning + Acting) loop powered by a locally-hosted LLM via Ollama to intelligently plan and execute multi-phase security assessments — from reconnaissance through exploitation.

It wraps industry-standard Kali Linux tools into a unified AI-driven interface, intelligently deciding which tool to use, and in what order, based on your natural-language objective in English or Thai.

The agent maintains a sliding window memory, queries a RAG knowledge base for pentest techniques, performs live CVE lookups, and includes safety controls to block dangerous tools without your explicit permission.

Key Features

What Makes LONLY Different

Nine core capabilities that transform a bare LLM into an autonomous security agent.

Autonomous ReAct Agent Loop

The LLM reasons, selects a tool, observes the output, then decides the next step — with safety checkpoints and user confirmations at critical decision points.

Safety Controls

Two-tier protection: intrusive tools (SQLMap, Nikto, enum4linux) are automatically blocked with a warning; tools like crackmapexec, hydra, and Metasploit require explicit user confirmation before execution.

Natural Language Interface

Give objectives in plain English or Thai; the agent handles the technical execution and tool orchestration.

RAG Knowledge Base

Augments the LLM with a local ChromaDB vector store of penetration testing cheat sheets and documentation.

BloodHound Analysis

Parses SharpHound collection ZIPs locally using NetworkX to find attack paths to Domain Admins.

Live CVE Lookup

Queries the NVD API in real time and cross-references results with the local Exploit-DB via SearchSploit.

Sliding Window Memory

Maintains a rolling chat history (last 20 messages) to preserve context without overflowing the LLM context window.

Risk-Budget Checkpoint

Tool calls accumulate risk points; when the budget (5 pts) is exceeded, the operator reviews progress and decides to continue, stop, or redirect.

Cross-Task Carryover

Fabrication/overclaim/placeholder events decay across tasks, preventing degraded LLM behavior from silently compounding.

Getting Started

Installation & Quick Start

Get LONLY running on your Kali Linux machine in minutes.

1. Clone the repository

bash 
$ git clone https://github.com/poppp334/Lonly_HARNESS.git
$ cd lonly-pentest-agent

2. Install Python dependencies

bash 
$ pip install -r requirements.txt

3. Pull the LLM model via Ollama

bash 
$ ollama pull gemma4:e4b

4. (Optional) Set up the RAG knowledge base

bash 
# Place your pentest cheat sheets / markdown files in the knowledge/ folder
# Then run the ingestion script
$ python ingest_knowledge.py

5. Run LONLY

bash 
$ python pentest_agent.py
